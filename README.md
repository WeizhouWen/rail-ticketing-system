# 高铁售票系统

高铁售票系统是一个简单的Spring Boot实战项目，提供铁路运行信息查询以及购票出行服务（模拟）。

## 项目用到的技术

项目采用前后端分离开发。 

* SpringBoot2.1.6
* Mybatis
* Mysql
* Redis
* druid
* mybatis generator
* HTML
* JQuery
* Bootstrap
